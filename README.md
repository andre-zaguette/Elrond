# Middle-Earth Agents

The central command and control repository for the agentic development workflow. This project orchestrates the collaboration between specialized agents to deliver high-quality, scalable software.

## The Fellowship
- **Gandalf (Orchestrator):** Strategic architecture, harness engineering, and task delegation.
- **Boromir (Quality Guardian):** Testing authority, quality gatekeeper, and autonomous verification agent.
- **Galadriel (UX/UI Architect):** Vision, usability, and design standards.
- **Elrond (Frontend Master - VueJS):** Vue.js component development and state management.
- **Celebrimbor (Master Smith - React/NextJS):** React/Next.js interface and performance optimization.
- **Radagast (Nature's Steward - Backend Python):** Python backend services and data integrity.
- **Beorn (Skin-Changer - Backend Node/NestJS):** Node.js/NestJS backend services and API robustness.

## The Palantír (Universal Archive)
- **Shared Wisdom:** The Palantír serves as the single source of truth for architecture, patterns, and shared interface contracts.
- **Inter-Agent Protocol:** Agents MUST consult the Palantír whenever they must exchange information or align on cross-functional tasks.

## Core Mandates
- **Context First:** All non-trivial changes require a `docs/contexto.md`.
- **Proof Before Alloy:** No code is merged without Boromir's verification ("Horn of Gondor").
- **Harness Integrity:** Every session must update `QUEST_PROGRESS.md`.

## Getting Started
Ensure you have the agent environment configured. Invoke Gandalf to map your next quest.
