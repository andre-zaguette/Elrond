# Middle-Earth Agents

The central command and control repository for the agentic development workflow. This project orchestrates the collaboration between specialized agents to deliver high-quality, scalable software across multiple environments.

## The Fellowship

- **Gandalf (Orchestrator):** Strategic architecture, harness engineering, and task delegation. Orchestrates the Fellowship's efforts.
- **Boromir (Quality Guardian):** Testing authority, quality gatekeeper, and autonomous verification agent. Ensures code integrity across all layers.
- **Galadriel (UX/UI Architect):** Vision, usability, and design standards. Specializes in cross-platform UI and component design.
- **Elrond (Frontend Master - VueJS):** Vue.js component development and state management. Builds modular frontend pieces.
- **Celebrimbor (Master Smith - React/NextJS):** React/Next.js interface and performance optimization. Crafts performant, type-safe UIs.
- **Radagast (Nature's Steward - Backend Python):** Python backend services and data integrity. Nurtures backend logic and services.
- **Beorn (Skin-Changer - Backend Node/NestJS):** Node.js/NestJS backend services and API robustness. Builds scalable backend systems.

## The Palantír (Universal Archive)

- **Shared Wisdom:** The Palantír serves as the single source of truth for architecture, patterns, and shared interface contracts.
- **Inter-Agent Protocol:** Agents MUST consult the Palantír whenever they must exchange information or align on cross-functional tasks.

## Core Mandates

- **Context First:** All non-trivial changes require a `docs/contexto.md`.
- **Proof Before Alloy:** No code is merged without Boromir's verification ("Horn of Gondor").
- **Harness Integrity:** Every session must update `QUEST_PROGRESS.md`.
- **Multi-Environment Agents:** All agents are designed to operate beyond a single CLI, functioning as independent modules or plugins.

## Getting Started

Ensure you have the agent environment configured. Invoke Gandalf to map your next quest.
